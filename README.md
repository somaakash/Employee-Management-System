# Employee Management System (CRUD)

This is a **Java Web Application** developed using **Java Servlets, JDBC, MySQL, and HTML**.  
The project demonstrates basic **CRUD operations** (Create, Read, Update, Delete) on employee data.

- Java **Servlets handle backend logic**

## âš™ How the Application Works

1. User opens **Home Page (index.html)**
2. Selects an option:
   - Add Employee
   - View Employees
   - Search Employee
3. HTML form sends data to a **Servlet**
4. Servlet connects to **MySQL using JDBC**
5. Data is inserted / fetched / deleted from database
6. Result is displayed in the browser
